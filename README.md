# Yahboom Song and Dance
Custom performance blocks for the Yahboom SmartCar.

## Performance Blocks
* **Fast Wiggle**: Makes the robot shake back and forth.
* **Figure S**: Drives in an 'S' shape.
* **Pop a Wheelie**: Sudden burst of speed to lift the front.

## Supported Targets
* PXT/microbit
